## 🧭 Why Helm?

Computer Science is a vast ocean. **Helm** is the dashboard every student needs to stay on course. It's not just another list of links; it's a curated experience built on the philosophy of **structure, clarity, and bold design.**

We believe that learning to code shouldn't feel like a chore. By combining **NeoBrutalist** aesthetics with practical, high-value data, we've created a home for the next generation of developers.

---

## ✨ Features

- **📍 Intelligent Roadmaps**: Comprehensive paths for Web, AI, Cyber, and Mobile. Stop guessing what to learn next.
- **📚 Gold-Standard Courses**: No filler. Only the best free and high-quality educational resources.
- **💼 Career OS**: A deep dive into landing internships, mastering interviews, and understanding the industry.
- **🛠️ Toolbox**: Essential developer software, VS Code extensions, and student benefit hacks.
- **📄 Portfolio & CV**: Craft a professional presence that gets noticed by recruiters.
- **📊 Local Progress Sync**: Check off tasks as you go. Your progress is saved locally in your browser.
---

## 🎨 Visual Style: NeoBrutalism

Helm is designed with a **NeoBrutalist** aesthetic, inspired by modern design trends:
- **High Contrast**: Bold black and white borders.
- **Vibrant Accents**: Sharp, flat color palettes.
- **Solid Shadows**: Chunky, offset shadows that give the UI a "pressed" physical feel.
- **Typography First**: Leveraging **Space Grotesk** for a geometric, futuristic vibe.

## 🚀 Tech Stack

- **Framework**: [Next.js 16+](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Font**: Space Grotesk & Inter

## 🛠️ Getting Started

### 1. Prerequisites
- **Node.js**: 18.x or later
- **Package Manager**: npm, yarn, or pnpm

### 2. Setup
```bash
# Clone the repository
git clone https://github.com/u-wz/helm.git

# Enter the directory
cd helm

# Install dependencies
pnpm install
```

### 3. Launch
```bash
pnpm run dev
```
Navigate to [http://localhost:3000](http://localhost:3000) to see the magic.
